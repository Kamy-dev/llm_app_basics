## 1. 機能

- Retrieval-Augmented Generation(RAG)を用い、アップロードしたPDF文書に記載された情報を元にLLMに回答させる。
- NotionAPIを利用し、LLMとの会話履歴をNotionに送る。
- LLMの回答の根拠となるドキュメントの該当部分をコンソールに出力させる。
